## 1. Selection Criteria
A Manus output is considered a "High-Value Candidate" if it meets at least one of the following:
- It is a structured Architecture Document (`.md`).
- It is a formal Registry (`.md` or `.json`).
- It is a Gate Report validating a system state.
- It is a JSON Schema defining data structures.
- It is an MPM used to orchestrate complex tasks.
