# Tool Intelligence Layer

## Definition
Operational knowledge layer mapping tools, apps, devices, services, workflows, integrations, decisions, capabilities, and routing rules.

## Architecture
- [[Tana]] = canonical typed graph
- [[Notion]] = readable dashboard
- [[Obsidian Canvas]] = human spatial graph UI
- [[n8n]] = maintenance automation
- [[ChatGPT]] = synthesis and K-card generation
- [[Manus]] = execution assistant

## Key views
- Tool by capability
- Tool by project
- Tool by device
- Tool by integration strength
- Tool by decision status
- Maintenance queue
