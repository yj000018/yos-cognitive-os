# Y-WORLD.net Vision

> Domain: Y-WORLD.net

Y-WORLD.net represents the public-facing portal, the dynamic cognitive atlas, and the agentic web runtime of Y-WORLD.

## 1. Core Functions
1. **Public Portal**: Human-facing showcase of projects, vision, and publications.
2. **Private Cockpit**: Secure web-accessible membrane to control smart home, agents, and workflows.
3. **Knowledge Atlas**: Interactive visual graph of public K-Cards.
4. **Symbolic Atlas**: Web-native portal for the Archetypes system.

## 2. Security Membrane
* **Private Vault**: All operational files, personal logs, and smart home configs remain secure inside the local Obsidian vault.
* **Public Membrane**: Only explicitly marked notes (e.g., `publish: true`) are synchronized and rendered on Y-WORLD.net.
