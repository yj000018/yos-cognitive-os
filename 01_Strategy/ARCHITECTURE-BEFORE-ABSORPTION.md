# Architecture Cognitive Avant Absorption Massive

> Canonical doctrine — DOCTRINE-YOS-002
> Last updated: 2026-07-03

## Principle

**No broad source acquisition, normalization, synthesis, or merge should happen before the cognitive architecture is defined and validated.**

This is the foundational discipline of KAP/Y-OS. It prevents the most common failure mode of knowledge management systems: importing everything before knowing what to do with it, resulting in an unnavigable, untrustworthy corpus.

## What is Forbidden Before Gate Authorization

- Broad source acquisition (bulk import of Obsidian, ChatGPT, Notion)
- Normalization of old source corpora
- Current-best synthesis generation
- Legacy repo merge
- Source mutation (modifying original sources)

## What is Permitted Now

- Metadata census (counting, listing, tagging sources without reading content)
- Architecture definition (this work)
- Control Plane bootstrap (this gate)
- Selective deep extraction of high-value durable outputs (e.g., Manus gate reports)
- Source cataloguing and registration
